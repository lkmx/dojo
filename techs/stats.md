Stats is a system monitor for the Mac menu bar. It allows to monitor:

- CPU utilization
- GPU utilization
- Memory usage
- Disk utilization
- Network usage
- Battery level
- Fan's control
- Sensors information (Temperature/Voltage/Power)
- Bluetooth devices
- Multiple time zone clock
